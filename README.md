# B.T Mtolo — Portfolio Website

A personal portfolio website for **Boikhantsho Tumelo Mtolo (B.T Mtolo)**, Web Developer & Junior Software Developer based in South Africa. The site showcases skills, projects, and contact information in a clean, modern single-page layout.

##  Live Sections

- **Hero** — Introduction, headline, and call-to-action buttons
- **About** — Background, personal statement, and quick stats (experience, projects, clients, dedication)
- **Skills** — Frontend, Development, and Client Work capabilities with tech pills
- **Projects** — Showcase of completed work (BOITHA HOLDINGS, BM DIGITAL, Login & Chat System)
- **Contact** — Email, LinkedIn, GitHub, and website links
- **Footer** — Closing note and copyright

##  Project Structure

```
my_portfolio/
├── my_portfolio.html        # Main HTML file
├── css_assets/
│   └── stylesheet.css       # Site styling (referenced, not included here)
├── script_assets/
│   └── script.js            # Site interactivity (referenced, not included here)
└── images/
    └── VEJE4034.PNG          # Profile photo
```

>  Note: The CSS (`css_assets/stylesheet.css`) and JS (`script_assets/script.js`) files are referenced by the HTML but were not included in this upload. Make sure both files exist in the correct folders relative to `my_portfolio.html`, or the page will be unstyled and non-interactive.

##  Built With

- **HTML5** — page structure
- **CSS3** — custom styling (via `stylesheet.css`)
- **JavaScript** — interactivity (via `script.js`), including a custom cursor effect
- **Google Fonts** — [Cormorant Garamond](https://fonts.google.com/specimen/Cormorant+Garamond) & [Jost](https://fonts.google.com/specimen/Jost)

##  Getting Started

1. Download or clone this project folder, keeping the folder structure intact.
2. Make sure `stylesheet.css` is placed inside `css_assets/` and `script.js` is placed inside `script_assets/`.
3. Place your profile photo inside the `images/` folder, matching the filename used in the HTML (`VEJE4034.PNG`) — or update the `<img src="">` path to match your own file.
4. Open `my_portfolio.html` directly in a browser, or serve it with a local dev server (e.g. VS Code's Live Server extension) for the best experience.
